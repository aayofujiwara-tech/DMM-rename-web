安全性について

ダウンロードされるスクリプトの動作と安全性の説明

安全性の概要

スクリプト生成場所ブラウザ上で完結・サーバー不関与
実行コマンドRename-Item（Windows）/ mv（Mac/Linux）のみ
スクリプト実行中のネット通信一切なし
ファイル操作ファイル名変更のみ・削除や上書きなし
動画ファイルへのアクセスなし（ファイル名のテキストのみ処理）

各ファイルのコード

rename.ps1 PowerShell
# DMM Renamer - リネームスクリプト (Windows) $folder = "C:\Users\...\Downloads" Rename-Item -Path "$folder\ipx00500hhb.dcv" -NewName "[IPX-500] 甘えたい夜に - 白石りな.dcv" -ErrorAction SilentlyContinue # ... 変換対象ファイル分だけ繰り返し ... Write-Host "完了しました" -ForegroundColor Green Read-Host "Enterキーで閉じる"
実行する.bat バッチファイル
@echo off chcp 65001 > nul powershell -ExecutionPolicy Bypass -File "%~dp0rename.ps1"
実行する.sh bash / Mac・Linux
#!/bin/bash # DMM Renamer - リネームスクリプト (Mac/Linux) FOLDER="/Users/.../Downloads" mv "$FOLDER/ipx00500hhb.dcv" "$FOLDER/[IPX-500] 甘えたい夜に - 白石りな.dcv" 2>/dev/null # ... 変換対象ファイル分だけ繰り返し ... echo "完了しました"

よくある疑問

実行されるコマンドは Rename-Item（Windows）と mv（Mac/Linux）のみです。 このページで全コードを確認できます。難読化は一切行っていません。 メモ帳・テキストエディタでファイルを開けばすべての処理内容を確認できます。
スクリプトはファイル名の変更のみ行います。削除・上書き・移動は行いません。 -ErrorAction SilentlyContinue により、 エラーが出ても他のファイルの処理は継続します。 念のため実行前に対象フォルダをバックアップしておくことをお勧めします。
スクリプトの実行中はインターネット通信を一切行いません。 変換時にFANZA公式APIへ品番（CID）を送信しますが、ファイル名の残り部分・動画内容は送信しません。 入力されたファイル名はサーバーに永続保存されません。
PowerShellスクリプト（.ps1）はWindowsのデフォルト設定で実行がブロックされます。

「実行する.bat」をダブルクリックすると -ExecutionPolicy Bypass フラグで自動的に回避されます。

「WindowsによってPCが保護されました」と表示された場合は「詳細情報」→「実行」をクリックしてください。
