このツールについて

DMM Renamer βプロトタイプ版

概要

DMMからダウンロードした動画ファイル（.dcv / .wsdcf）のファイル名を、 女優名・タイトルを含むわかりやすい名前に変換するWebツールです。

ブラウザ上で動作し、FANZA公式アフィリエイトAPI（v3）を使ってタイトル・女優名を取得します。 変換後のリネームはダウンロードしたスクリプトをローカルで実行することで完結します。

処理の流れ

1
ファイル名から品番（CID）を抽出

正規表現でファイル名を解析し、品番コードを取り出します。

ipx00500hhb.dcv → CID: ipx00500 / ラベル: IPX-500
2
FANZA公式 Affiliate API v3 に問い合わせ

取得したCIDでFANZAの公式APIを呼び出し、タイトル・女優名を取得します。

https://api.dmm.com/affiliate/v3/ItemList
3
新しいファイル名を構築

取得したタイトル・女優名を組み合わせて新しいファイル名を生成します。

[IPX-500] 甘えたい夜に - 白石りな.dcv
4
リネームスクリプトをZIPでダウンロード

PowerShell（Windows）・bash（Mac/Linux）用スクリプトをブラウザ上で生成し、ZIPで提供します。サーバーは関与しません。

やっていること

  • ✅入力されたファイル名から品番（CID）を抽出
  • ✅FANZA公式API（Affiliate API v3）でタイトル・女優名を取得
  • ✅リネームスクリプト（.ps1 / .sh / .bat）をブラウザ上で生成
  • ✅すべての処理はブラウザとFANZA公式サーバーの間で完結

やっていないこと

  • ❌ファイル本体（動画）のアップロード・取得
  • ❌入力されたファイル名のサーバーへの永続保存
  • ❌個人情報・IPアドレスの収集・保存
  • ❌Cookieの発行
  • ❌第三者へのデータ提供
  • ❌スマートフォン・タブレットへの対応
