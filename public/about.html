このツールについて

DMM Renamer βプロトタイプ版

概要

DMMからダウンロードした動画ファイル（.dcv / .wsdcf）のファイル名を、 女優名・タイトルを含むわかりやすい名前に変換するWebツールです。

ブラウザ上で動作し、FANZA公式アフィリエイトAPI（v3）を使ってタイトル・女優名を取得します。 変換後のリネームはダウンロードしたスクリプトをローカルで実行することで完結します。

やっていること

  • ✅入力されたファイル名から品番（cid）を抽出する
  • ✅FANZA公式API（Affiliate API v3）でタイトル・女優名を取得する
  • ✅リネーム用スクリプト（.ps1 / .sh / .bat）を生成してダウンロードさせる
  • ✅すべての処理はブラウザとFANZA公式サーバーの間で完結する

やっていないこと

  • ✖️ファイル本体（動画）のアップロード・取得
  • ✖️入力されたファイル名のサーバーへの永続保存
  • ✖️個人情報・IPアドレスの収集・保存
  • ✖️Cookieの発行
  • ✖️第三者へのデータ提供
  • ✖️スマートフォン・タブレットへの対応
